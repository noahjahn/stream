# Stream

Discord wants you to pay for quality screen sharing. Fuck that.

Get started: `docker compose run --rm node npm install && docker compose up`

## Ideas

### Ref

- https://webrtc.github.io/samples/
- https://developer.mozilla.org/en-US/docs/Glossary/WebRTC

### Discovery

- Be able to configure fps and resolution: optimize for games vs optimize for text
- Provide recommendations for share and receive settings based on client bandwidth
- Configure own P2P broker server
- Video calling in addition to screen sharing
- Better call UI
- Voice calling
- End to End encryption
